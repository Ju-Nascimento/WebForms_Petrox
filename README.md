📦 Sistema de Controle de Estoque – ASP.NET Web Forms

Este projeto foi desenvolvido como parte da minha preparação para uma entrevista de estágio no setor de TI da Petrox. O objetivo foi praticar e demonstrar conhecimentos em desenvolvimento web utilizando as mesmas tecnologias utilizadas pela equipe da empresa.

A aplicação consiste em um sistema simples de controle de estoque, permitindo gerenciar produtos e registrar movimentações de entrada e saída.

🚀 Tecnologias Utilizadas

ASP.NET Web Forms

.NET Framework 4.8

C#

SQL Server

Bootstrap 5

ADO.NET

A escolha do Web Forms foi intencional, pois essa é a tecnologia utilizada pela equipe de desenvolvimento da empresa, permitindo demonstrar familiaridade com o stack existente.

🎯 Objetivo do Projeto

Criar um sistema simples de gerenciamento de estoque que permita:

Cadastro de produtos

Consulta de produtos cadastrados

Registro de movimentações de estoque

Atualização de informações

Controle da quantidade disponível

Além de reforçar conceitos como:

Integração entre ASP.NET e banco de dados

Estruturação de CRUD completo

Organização de código em camadas

Manipulação de GridView e controles Web Forms

Estilização utilizando Bootstrap

⚙️ Funcionalidades
📋 Cadastro de Produtos

Permite registrar novos produtos no sistema com informações como:

Nome do produto

Categoria

Quantidade em estoque

Outras informações relevantes

🔎 Consulta de Produtos

Os produtos cadastrados podem ser visualizados em uma GridView, permitindo:

Listagem dos produtos

Visualização das informações cadastradas

Acesso às opções de edição

✏️ Alteração de Produtos

Permite atualizar os dados de um produto existente no sistema.

📦 Movimentação de Estoque

O sistema permite registrar movimentações de estoque, como:

➕ Entrada de produtos

Registro de novas unidades adicionadas ao estoque.

➖ Saída de produtos

Registro de retirada de itens do estoque.

O sistema atualiza automaticamente a quantidade disponível após cada movimentação.

🗄️ Banco de Dados

O projeto utiliza SQL Server para armazenamento das informações.

Principais entidades:

Produtos

Movimentações de estoque

O acesso ao banco foi feito utilizando ADO.NET, realizando operações como:

INSERT

SELECT

UPDATE

DELETE

🎨 Interface

A interface foi construída utilizando:

ASP.NET Web Forms

Bootstrap 5

Isso permite uma aplicação com:

Layout organizado

Responsividade básica

Melhor experiência visual


Uso de Bootstrap em aplicações Web Forms

Organização da lógica de aplicação em C#
